# Hardware Stage H9 Full-layer Results

Status: PASS for the implemented interleaved full-layer RTL matrix.

Command:

```text
make hw-h9-rtl-sim
```

The H9 RTL script compiles the Stage 7D full `transformer_layer` bit-exact
testbench with:

```text
ATTENTION_PE_ARCH=PAPER_ARRAY
ATTENTION_SCHEDULE=INTERLEAVED
```

Configured runs:

| Run | N_HEAD | D_HEAD | Tokens | Result |
|---|---:|---:|---:|---|
| h9_layer_h1_d8 | 1 | 8 | 1 | PASS |
| h9_layer_h2_d8 | 2 | 8 | 1 | PASS |
| h9_layer_h2_d8_two_token | 2 | 8 | 2 | PASS |
| h9_layer_h4_d8 | 4 | 8 | 1 | PASS |
| h9_layer_h2_d16 | 2 | 16 | 1 | PASS |

The Stage 7D testbench checks the full order:

```text
RMSNorm1 -> Q/K/V projection -> interleaved Attention -> W_O -> residual1
-> RMSNorm2 -> W1 -> ReLU -> W2 -> residual2 -> final output -> layer done
```

Expected final FP32 outputs are generated by the Stage 7D full-layer reference
and compared bit-for-bit in the RTL testbench. The two-token H2/D8 run verifies
KV growth through the full wrapper.

Limit: full-layer reset injection at every requested micro-stage is not yet
implemented.
